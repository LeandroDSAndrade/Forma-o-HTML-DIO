HTML: Um breve resumo

HTML é uma linguagem de marcação amplamente utilizada nas comunicações Web. Toda e qualquer página na internet está escrita sob um arquivo .html. Entretanto dificilmente vemos uma página Web "crua". Encontramos esse tipo de arquivo associada com a linguagem de marcação CSS e com a linguagem de programação JavaScript

Para este resumo iremos trazer as seguintes tags do HTML, adicionadas de uma explicação e um exemplo:

  • tag h
  • tag p
  • tag strong
  • tag small
  • tag i
  • tag u
  • tag de listas
  • tag de âncora
